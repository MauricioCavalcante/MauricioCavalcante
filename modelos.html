Quando há necessidade de se obter dados dos usuários, para cadastro por exemplo, os formulário precisam ser práticos e com a menor margem de erros possíveis, para isso a configuração de entrada de dados estando bem tratadas, geram conforto e praticidade tanto para os usuários que preenchem quanto para quem recebe esses dados seja no banco de dados, via planilha (Sim, é possível enviar os dados diretamente para uma planilha, caso não seja um sistema que precise de um banco de dados, como o SQL) ou mesmo via e-mail.

Uma das coisas que posso oferecer é o mascaramento do número de telefone, para padrões nacionais +55 "(XX) XXXXX-XXXX", ou qualquer outro padrão internacional, exemplo +1 "(XXX) XXX-XXXX", gerados automaticamente ao selecionar o páis de origem do número de telefone.
